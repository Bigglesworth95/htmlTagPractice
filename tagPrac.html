HTML Tags Cheat Sheet

Headings

Headings from h1-h6 identify blocks & sections Every page must have one h1.

Paragraphs

Paragraphs of text go in p tags

Lists

Unordered Lists

Unorered lists are marked with bullet points and go in 'ul' tags. Each list item goes in a li tag.

Types of lists:

  • Unordered lists (ul)
  • Ordered lists (ol)

Ordered Lists

Ordered lists are nunmbered and go in ol tags. Each list item goes in a li tag.

Polya's Problem Solving Framework

  1. Understand the problem
  2. Come up with a plan.
  3. Carry out the plan
  4. Go back and improve your solution

Anchors (links)

Anchor tags create links to other pages. The URL goes in the href attribute. Go to DuckDuckGo to search for more information.

Setting 'target=_blank" will open the link in a new window. See: MDN for more details.

Anchor tags can also create links to a new location on the same page. Instead of putting a URL in the href attribute, just use the 'id' of the element you want to navigate to. This allows you to Jump to the top of the page
